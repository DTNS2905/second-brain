# 01 - Interface

## The Layout

```
┌─────────────────────────────────────────────────────┐
│  Left Sidebar  │        Editor          │ Right Sidebar│
│                │                        │              │
│  File Explorer │   Your note content    │  Backlinks   │
│  Search        │   (Markdown)           │  Properties  │
│  Bookmarks     │                        │  Outline     │
│  Tags          │                        │              │
└─────────────────────────────────────────────────────┘
```

## Key Shortcuts

| Action | Shortcut |
|--------|----------|
| New note | `Cmd+N` |
| Open note | `Cmd+O` |
| Command palette | `Cmd+P` |
| Search vault | `Cmd+Shift+F` |
| Toggle left sidebar | `Cmd+\` |
| Graph view | `Cmd+G` |
| Toggle edit/read mode | `Cmd+E` |

## Two Modes

**Editing mode** — you type Markdown syntax directly.
**Reading mode** — Obsidian renders the Markdown as formatted text.

Toggle between them with `Cmd+E` or the button in the top-right corner.

## The Command Palette

Press `Cmd+P` — this is the most powerful shortcut in Obsidian.
Type anything: *"daily note"*, *"graph"*, *"theme"* — it finds and runs any action.
