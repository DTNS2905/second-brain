## Community Plugins

Settings → Community Plugins → Browse

**Essential plugins most people install:**

| Plugin | What it does |
|--------|-------------|
| **Dataview** | SQL-like queries before Bases existed — still powerful |
| **Templater** | Advanced templates with logic, dates, user input |
| **Calendar** | Visual calendar for daily notes |
| **Excalidraw** | Hand-drawn diagrams inside notes |
| **Kanban** | Kanban boards from Markdown lists |
| **Linter** | Auto-formats notes on save |
| **Various Complements** | Autocomplete for words/links as you type |

> [!warning] Install carefully
> Only install plugins with many downloads and recent updates.
> Community plugins have full access to your files.

## Useful Workflows

### Zettelkasten (Atomic Notes)

One idea per note. Link ideas instead of organizing by topic.

```
Closure (JS).md  →  links to →  Scope.md
                               ↓
                          First-class Functions.md
```

Rules:
- Notes are small (one concept)
- Every note links to at least one other
- No folders — only links and tags

---

### PARA Method

Organize by **actionability**, not topic.

```
Vault/
├── Projects/   ← active, has a deadline
├── Areas/      ← ongoing responsibility (health, work)
├── Resources/  ← reference material
└── Archive/    ← inactive items
```

---

### Daily Note System

Use daily notes as your **inbox**. Process them weekly.

```
Daily/2026-06-03.md  →  captures raw thoughts
        ↓ weekly review
Permanent notes in Projects/ or Resources/
```

## Tips From Power Users

> [!tip] Link first, organize later
> Don't spend time making folders. Write notes and link them.
> The structure will emerge naturally.

> [!tip] The 2-minute rule
> If a thought takes less than 2 minutes to capture, write it now.
> Use your daily note as an inbox.

> [!tip] Review your graph monthly
> Open Graph View (`Cmd+G`) and look for isolated notes — orphan notes with no links.
> They're either useless or need to be connected.

> [!tip] Use properties for anything you want to filter
> If you ever want to ask "show me all X where Y = Z", that Y needs to be a property.
